## Overview
This repository contains the computational framework, data analysis scripts, and analytical models developed for my research on laser-plasma interactions, specifically focusing on stron laser plasma interaction with QED. 

The codebase is divided into two main branches:
1. **Analytical Solvers**: Numerical integration of the Green's functions for the 1D wave equation to model time-domain reflection and transmission of electromagnetic pulses in localized plasmas.
2. **PIC Simulation Diagnostics**: Data processing pipelines for the OSIRIS Particle-in-Cell (PIC) code. It includes precise energy conservation algorithms, Poynting flux integration, and phase-space analysis.

## Key Modules

### 1. Analytical Green's Function Solver
A Python implementation to replicate and extend theoretical models of plasma mirrors (e.g., Nodland's solutions). 
* **Features**: 
  * Convolution-based boundary condition evaluation.
  * Retarded time propagation and dispersive wakefield integrals.
  * Evaluation of arbitrary pulse envelopes (Gaussian, polynomial).

### 2. OSIRIS Data Analysis Pipeline
Scripts designed to process HDF5 (`.h5`) and `HIST` files generated by OSIRIS 1D/2D simulations.
* **Features**:
  * `energy_integrator.py`: Absolute Reflectance ($R$) and Transmittance ($T$) calculation via temporal integration of the Poynting flux ($S_1$) at simulation boundaries, eliminating spatial truncation errors.
  * `hist_visualization.py`: Evaluation of global energy conservation tracking electromagnetic field energy vs. plasma kinetic energy over time.
  * Spatial field animations and phase-space grid-heating diagnostics.
